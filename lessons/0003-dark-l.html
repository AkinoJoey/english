Lesson 03 · Listening

消えたLを探す

あなたの間違いは偶然ではありませんでした。2つとも同じ音が犯人です。

このレッスンは、あなたの実際の誤答から設計されています。 一般論ではありません。

1. 犯人の特定

前回、あなたが間違えた箇所と、実音源で詰まった箇所を並べます。

正解あなたが聞いた音消えたもの
a couple ofa cup ofl
actualelectricl

両方とも、語末の L が原因です。そして両方とも、あなたの耳が悪いわけではありません。その L は、実際に子音として鳴っていないからです。

英語の /l/ には2つの顔があります。

Light L [l]Dark L [ɫ]
位置音節の頭（母音の前）音節の末尾（母音の後）
例list, load, believenull, call, actual
聞こえ方ラ行として認識できるウ / オ のような母音

dark L は音声学的に「light L と軟口蓋接近音 [ɰ] を同時に出した音」と記述されます。つまり子音と母音の中間で、しばしば完全に母音化します（Dark L, University of Manitoba）。

聞き比べてください。同じ文字なのに、別の音です。

2. 本当の犯人は、耳ではなく記憶

ここが今日いちばん重要な部分です。

あなたが actual を聞き逃したのは、知らない語だったからではありません。知っている語です。問題は、あなたが記憶している形が違うことです。

語記憶している形実際の音
actualアクチュアル（5拍）/ˈæk.tʃu.əɫ/ アクチュオ（3音節）
coupleカップル（4拍）/ˈkʌ.pəɫ/ カポゥ（2音節）

あなたの脳は末尾の「ル」を待っていた。しかし来なかった。だから照合に失敗し、couple は cup に切り詰められ、actual は形の似た別語（electric）に化けた。

これはカタカナ語の呪いです。カタカナは dark L に存在しない母音を足します。そして皮肉なことに、知っているカタカナ語ほど危険です。知らない語なら素直に音を聞きますが、知っている語は「知っている形」で照合してしまうからです。

診断手順が3つ目に増えます: 聞き取れない語に出会ったとき —— ①境界が消えた（linking）か? ②機能語が潰れた（reduction）か? そして今日 ③末尾の L が母音化したか?

3. 練習 — 聞いて、打ち込む

前回同様、答えを見る前に必ず一度は入力してください。

(1) 復習 — 前回の弱点から

まず can / can't をもう一度。前回間違えた箇所です。t を探さないでください。母音の長さで判断します。

否定の can't。母音が長く、そこで一瞬止まります。加えて locally の l はlight L（音節の頭）なので、こちらははっきり聞こえたはずです。

肯定の can です。/kən/ と潰れ、ほぼ「クン」。直前の問題と母音を比べてください —— この対比が判別軸そのものです。

(2) 今日の本題 — 消えた L

前回間違えた couple の再登場です。加えて tools（トゥオズ）と available（アヴェイラボゥ）。3語すべて末尾が母音のように聞こえるのが正常です。

実音源であなたが詰まった actual です。アクチュオ。value（ヴァリュ）、null（ナォ）。カナで「アクチュアル・バリュー・ヌル」と記憶していると、まず捕まえられません。

pull（プォ）、model（マドォ）、file（ファイオ）、shell（シェォ）。毎日使う語ばかりなのに、音は全部カタカナと違います。

local は light L と dark L が1語に同居する好例です。頭の l ははっきり、末尾の l は母音化して「ロウコゥ」。同じ文字が別の音になる —— これが異音（allophone）です。

(3) 実音源の復元

あなたが詰まった実際の文です。今なら聞き取れるはずです。

clone の l は light L（はっきり）、pull と actual の l は dark L（母音化）。同じ文の中で L が別の音になっていることを確認してください。これが聞き分けの完成形です。

4. 口を使って定着させる

dark L を出すコツは舌先を歯茎につけないことです。日本語の「ル」は舌先が当たりますが、dark L は舌の奥を持ち上げるだけで終わることが多い。

手順: null と言うとき、「ナ」の後に「オ」と言うつもりで舌の奥を上げ、舌先は動かさない。それで正解に近づきます。

語やること音声
null → call → tool舌先を当てず「オ」で終える
couple → module → available末尾を「ポゥ / ジュオ / ボゥ」に
local頭ははっきり、末尾は溶かす

今日の勝ち: 「知っているはずの語が聞き取れない」現象の正体が分かりました。耳の問題ではなく、記憶している形の問題です。そして最も危険なのは知らない単語ではなく、カタカナで知っている単語だと分かりました。
