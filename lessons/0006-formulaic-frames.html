Lesson 06 · Listening

解読しない区間

英語の話し言葉には、聞き取ってはいけない場所があります。

1. 「リターンズ」の正体

前後を教えてもらって、確定しました。

What this basically does is it installs a GitHub…

あなたが聞いた「リターンズ」を分解します。両端だけが本物でした。

リター・ンズ
実在した音存在しない音実在した音
-cally の /l/脳が補完したdoes / is の /z/

音の端は正しく捉え、中央が薄くなった。そこに脳が「知っている語」を流し込んで、両端の辻褄が合う returns が出てきた —— これは音韻修復（phoneme restoration）と呼ばれる、正常な脳の働きです。欠けた音を語彙知識で埋める機能で、非語より実在語で強く起きます（Essentials of Linguistics 13.4）。

そして選ばれた語が returns だったのは偶然ではありません。 あなたは What this … does is という枠を正しく捉え、「ここには機能を説明する動詞が来る」と正確に予測していた。予測は当たっていて、候補の絞り込みだけが外れた。コーディング動画でその枠に入る最頻語は、確かに returns です。

前回の私の仮説は当たっていましたが、言い方を訂正します。これは失敗ではなく、新しい機能が動き始めた音です。レッスン 05 まで、あなたは予測を使っていませんでした。今回初めて使い、まだ精度が粗い。それだけです。

2. その区間には、そもそも情報がない

問題の箇所をもう一度見てください。

区間語数情報量
What this basically does is it6語ほぼゼロ
installs a GitHub…3語全部ここ

これは pseudo-cleft（擬似分裂文）という構文で、話し言葉に偏って現れます（Cleft sentence）。What X does is Y の形で、Y を強調するためだけに前半を置く。前半は意味を運んでいません。助走路です。

だから話者は前半に強勢の予算を割きません。潰れて当然の区間だった。あなたが聞き取れなかったのではなく、聞き取れる情報が入っていなかった。

数字で見る

話し言葉の pseudo-cleft に現れる動詞は、do が66%、happen が13%、say が8%。この3語で87%を占め、その前半部分は定型断片（formulaic fragment）として機能する——つまり後続の内容を「動作の報告」「出来事」「言い換え」に分類する標識にすぎません（Pseudo-cleft-like structures in conversation）。

さらに広く、話し言葉は68.81%が定型表現です（書き言葉の学術散文は21%）。出典

話し言葉の約7割は、解読する対象ではなく、見覚えのある形として認識する対象です。

3. 技術: is を軸として使う

pseudo-cleft には必ず軸（pivot）があります。is です。

What this basically does is it installs a GitHub app

手順: 文頭で What… が聞こえたら、意味の解読をやめて is を待つ。is が来たら解読を再開する。情報は100%そこから後ろにあります。

これはサボりではありません。母語話者も同じことをしています。 彼らは前半を1語ずつ処理せず、枠として認識して読み飛ばしている。あなたが1語ずつ拾おうとしていたのは、母語話者より重い処理をしていたということです。

覚えるべき枠（技術トーク頻出）

枠軸聞く
What this does is …is
What happens is …is
What I'm going to do is …is
All you have to do is …is
The way this works is …is
The thing is …is

basically / essentially / actually / obviously / literally は、この枠に自由に挿入されます。4音節あっても内容語ではありません。 見た目に騙されないでください。

5. レッスン 05 の訂正

前回「ラ行に聞こえたら /t/ に戻せ」と教えましたが、枝が1本足りませんでした。今回の -cally の「リ」がその証拠です。

ラ音の出どころ条件例
flap した /t/ /d/母音間 + 後ろが非強勢water, it occurs
light /l/母音の前（語頭・音節頭）basically, list

日本語のラ行は、英語の3つの別々の音をまとめて受け止めてしまいます。切り分けは音だけでは無理で、位置と強勢から絞り込みます。 詳細はリファレンスに追記しました。

実地練習（今週）: あの動画を字幕なしで30秒。今回は狙いを絞ります。
