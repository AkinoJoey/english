Lesson 05 · Listening

ラ行に化ける /t/

「イリグッズトゥミー」は、間違いではありません。あれは正確な音写です。

1. あなたの書いたものを分解する

it occurs to me がイリグッズトゥミーに聞こえた。並べてみます。

あなたが聞いた実際の音綴り
イɪi
リɾt（it の語末）
グəkoc
ッズɜːzcurs
トゥtəto
ミーmiːme

取りこぼしはひとつもありません。 6つの音の塊すべてが、実際に鳴っている音と対応しています。to が「トゥ」に潰れているのも正しく捉えている（レッスン 02 の弱形です）。

3回連続で同じ構造です。 レッスン 04 の「トゥーオーサー」、「パクチー」、そして今回の「イリグッズ」。あなたは音を正確に聞いている。足りないのは、聞こえた音を綴りに戻す変換規則だけです。今日はその規則のうち、最も出現頻度の高い1本を渡します。

2. 規則: /t/ はラ行になる

アメリカ英語の /t/ は、条件が2つ揃うと有声歯茎はじき音 [ɾ] に変わります。これを flapping（または t-voicing）と呼びます。

① /t/ が母音と母音に挟まれている
② 後ろの母音が非強勢である
出典: Flapping / Flapping in North American Pronunciation (JLLS)

water, better, city, writing —— すべて条件を満たすので、ラ行で鳴ります。

一方 attack, return, hotel は後ろに強勢があるので、/t/ ははっきり破裂します。

ここが今日の核心

この [ɾ] は、日本語のラ行の子音とほぼ同じ音です。英語母語話者に日本語の「ら」を聞かせると、彼らはそれを「英語の flap /t/ の実例」として分類します（Perception of English /r/ and /l/ by Japanese speakers）。

つまりこの音は、あなたが生まれたときから持っている音です。だから「イリ」と正確に書けた。問題は能力ではなく、ラ行 → /t/ という戻し方を知らなかったことだけです。

3. 語をまたいで起こる

リスニングで効くのはここです。語末の /t/ + 次の語の母音でも条件は揃います。しかも、その /t/ を持っているのはたいてい機能語です。

綴り実際の音聞く
it occursイリカーズ
set it upセリラップ
get it outゲリラウト
a lot of itアラーラヴィッ
what I didワライディッ

it / at / that / but / not / get / what / out / about —— この9語を覚えておけば、語をまたぐ flap のほとんどを説明できます。

4. 分節の道具として使う

前回は「強い音節を語頭とみなす」を学びました。flap はその裏側から効きます。

条件②により、ラ音の直後は必ず非強勢です。非強勢の音節は内容語の頭になりません。つまりラ音が聞こえたら「ここは語頭ではない」と確定できる。次の強い音節まで待てば、そこが内容語の先頭です。

「イリグッズ」で言えば —— リの直後の「グ」は非強勢（oc）。強いのは「ッズ」（CURS）。だから内容語は occurs ひとつ、そしてこれは前回の install と同じ、語頭に強勢が来ない例外語です（oc·CURS）。

5. 練習

(1) 規則を確かめる

flap（ラ行化）が起きるのはどちらですか。

ラ音が聞こえたとき、その直後の音節について確実に言えることは。

「イリグッズ」の「リ」の正体は何でしたか。

(2) あなたの一文を、解き直す

同じ音、同じ速さ。変わったのはラ行を /t/ に戻せるようになったことだけです。

(3) 語をまたぐ flap

強い音節は SET / UP / LO / MA。ラ音は set-it と it-up の2箇所で、どちらも it の前後です。

セリラップ の中に it が隠れています。ラ音に挟まれた弱い母音は、ほぼ必ず機能語です。

(4) 既習事項との混合

pull と model の dark L が母音化し、pull-it と it-out で flap が起きます。それでも強い音節は4つだけ。

レッスン 03 で the model を「demand」と融合させた文です。dark L・弱形・flap・分節が全部同時に効いています。これが実音源で起きていることの全体像です。

au·to·MA·tic の to も flap します。語の内側と語のまたぎで、同じ規則がまったく同じように働いています。
