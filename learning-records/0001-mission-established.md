# 0001. ミッションの確立と、最初の攻め筋の決定

- 日付: 2026-07-19
- ステータス: 有効

## 文脈

英語学習ワークスペースの初回セッション。既存の学習記録はゼロ。ヒアリングの結果:

- 目的は複数（技術ドキュメント読解 / 海外転職 / 日常会話）
- 自己申告レベルは**中級**
- **4技能すべて**に課題を感じている

「全部が弱い」という申告は、着手点の決定を難しくする。何から手を付けるかが最初の意思決定になった。

## 決定

**音の認識（connected speech）、特に linking から始める。**

## 理由

1. **転移が最大**。音の認識はリスニングとスピーキングの両方に効く。片方だけを鍛える施策より投資効率が高い。
2. **「中級で全技能が弱い」の典型的な正体**。読めるのに聞けない・話せないという症状は、語彙や文法ではなく音韻処理の欠落であることが多い。学習者は citation form で語を記憶しているが、実際の発話は connected speech で届くため照合に失敗する。
3. **難易度の順序が文献で裏付けられている**。linking は reduction / assimilation より処理しやすい（[JETAL](https://jurnal.uhn.ac.id/index.php/jetal/article/view/2051)）。ZPD の入口として適切。
4. **技術文脈に紐付けやすい**。`check it out` `roll it back` `is it up` など、実務頻出フレーズがそのまま教材になる。ミッションと直結する。

## 非自明な洞察

- ユーザーの「4技能すべてが弱い」は、**4つの独立した問題ではない可能性が高い**。音の土台が抜けていると、リスニングが崩れ、スピーキングが崩れ、結果としてリーディングも音声化できず遅くなる。単一の根本原因を疑うべき。
- したがって、**技能ごとに並行して教えるのは誤り**。土台を1つ直して、他技能への波及を観測する方針を取る。

## 未検証の前提

- ユーザーが本当に中級か（自己申告のみ。次回、実際の聞き取りテストで較正したい）
- 1日の学習可能時間が不明（NOTES.md に記録済み）
- 転職の目標時期が不明。決まればバックキャストで計画を組める

- 苦戦した → linking の練習量を増やし、実音源（ポッドキャスト）へ橋渡しする
