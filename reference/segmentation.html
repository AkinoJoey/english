Segmentation

音は聞こえている。問題は、どこで切るか。

大原則: 英語話者は強い音節を新しい語の先頭とみなして音の流れを切る（Metrical Segmentation Strategy, Cutler & Norris 1988）。英語の語の約90%が語頭強勢なので、この方略が機能する。出典: The role of strong syllables in segmentation for lexical access

なぜ日本語話者に効かないのか

この方略は強勢基準の言語（英語・ドイツ語・オランダ語）で有効で、モーラ基準の日本語には当てはまらない（Encyclopedia of Language Development: Metrical Segmentation）。

日本語（母語）英語（目標）
単位モーラ（拍）強勢音節
切り方等間隔の拍で刻む強い音節を語頭の目印にする
手がかりすべての拍が等価強い音節だけが目印

つまり日本語話者は、持っていない方略を新しく獲得する必要がある。耳の良し悪しの問題ではない。

切り方の手順

手順やること
1強い音節だけを拾う。弱い部分は無視してよい
2強い音節の直前に切れ目を入れる（そこが語頭）
3拾った強い音節から内容語を復元する
4内容語の間に入る機能語を文法から推測する

4番が重要。 機能語は聞き取るものではなく、推測するもの。潰れて鳴っていない以上、耳で捕まえようとしても無理がある。内容語さえ拾えれば、the / a / of / are は文法が決めてくれる。

実例 — 強勢だけで復元する

実際の文拾える強い音節復元
A couple of tools are available. COU… TOOLS… VAIL… couple / tools / available
Pull the model file from the shell. PULL… MO… FILE… SHELL pull / model / file / shell
Install the following packages. STALL… FOL… PACK… install / following / packages
Let's install it on a local terminal. STALL… LO… TER… install / local / terminal

語頭強勢でない例外

約10%は語頭強勢ではない。多くは接頭辞つきの動詞。
