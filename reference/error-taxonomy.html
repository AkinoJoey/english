誤り分類表

聞き取れなかった箇所を、既習の規則に振り分けるための1枚。

使い方。 transcript と照合して誤りを見つけたら、下の表でどの行に当たるかを言葉にしてから次へ進みます。分類できた誤りは既習の規則の適用漏れ —— 回数を重ねれば消えます。分類できない誤りだけが新しい情報で、それが次のレッスンになります。

2. 分類の順番

上から順に当てはめるのではなく、まず規模で分けます。

  1. 区間が丸ごと消えた／1語に化けた → 枠。速くて平坦な区間は情報を運びません。分節しようとせず枠ごと処理します
  2. 語数は合うが境界がずれた → 分節。強い音節から切り直します
  3. 1つの音が別の音に聞こえた → リンキング / 弱形 / dark L / ラ音。音素レベルの逆変換です
  4. 音は合っているのに語にならなかった → 階層。着地先の選択ミスです

音写が音素照合できないときは、分類してはいけません。 聞こえた形と実際の語で音素が対応しないなら、それは音写ではなく脳の補完結果です（basically does → リターンズ）。この場合は前後1文を添えて持ってきてください。音韻修復の解説

3. 一回聴きのプロトコル

実世界に巻き戻しはありません。練習も条件を寄せます。ただし、時計を2つに分けます —— 知覚を診断する単位と、口に入れる単位は別です（L09）。

診断の単位は「一息の句」（≦2秒・7音節）です。 音韻ループが逐語で保持できるのは約2秒まで（Baddeley et al. 1975）。これを超える区間を丸ごと書き取ると、音は消えて意味からの再構成になり、聞き逃したのか保持しきれなかったのか区別できなくなります。知覚を測るときは、必ず句に割ってから。15〜20秒は shadowing 専用の単位です。

手順やること狙い
115〜20秒の区間を選ぶ（transcript はまだ見ない）shadowing 用の単位
2知覚を試すときは、区間を一息の句に割る（息継ぎ・意味の切れ目が境目）音韻ループに収める（≦2秒）
31句を1回だけ再生し、書けるだけ書く純粋な知覚の負荷
4transcript と照合する誤りの特定
5誤りを上の表で分類する既習の規則を能動的に引き出す
6全句を終えたら、15秒を丸ごと shadowing する口に入れて検索範囲を広げる

速度は落としません。 ゆっくり再生は理解を助けますが、鍛えるべきは実速度での処理です。句単位で取れなければ、それは今日の知覚の穴として記録すればよく、埋めるのは transcript の仕事です。

4. 素材の選び方 —— narrow listening

話者と話題を固定し、素材だけを変えます。 Krashen (1996) の narrow listening です。同じ語彙・同じ言い回しが何度も回るので語彙と背景知識の負荷が下がり、音韻処理だけが残ります —— 鍛えたい部分だけに負荷が集中する。

固定するもの変えるもの
話者（同じ YouTuber / 同じポッドキャストのホスト）回・エピソード
話題領域（例: フロントエンド、インフラ）個別のトピック
形式（チュートリアル / 対談）—

目安は同じ話者・同じ領域で4〜5本。1本を極めるのではなく、似た素材を横に並べて量を通すのが要点です。1本の反復だけでは、聴き取りではなく暗記になります。
