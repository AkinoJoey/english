Formulaic Frames

話し言葉の約7割は定型。解読せず、形として認識する。

大原則: 話し言葉は 68.81% が定型表現（書き言葉の学術散文は21%）。出典
定型区間は情報を運ばないので、話者は強勢を割かず、潰れる。聞き取れないのではなく、聞き取る情報がない。

Pseudo-cleft（擬似分裂文）

What X does is Y の形。Y を強調するためだけに前半を置く。話し言葉に偏って現れる（Cleft sentence）。

使われる動詞は do 66% / happen 13% / say 8% で3語が87%。前半は後続内容を「動作」「出来事」「言い換え」に分類する標識にすぎない（Pseudo-cleft-like structures in conversation）。

手順

  1. 文頭で What / The thing / All you / The way が聞こえる
  2. 解読を停止し、軸 is を待つ
  3. is が来たら解読を再開
  4. 情報は100%その後ろにある

助走路は is の1語先まで

軸の後ろで主語が代名詞として繰り返されることが多い。これも情報を運ばない。

枠軸反復代名詞内容
What this basically doesisitinstalls a GitHub app
What I recommend you doisyougo to the top here
What I'd doisI'dcache it at the edge

※ このパターンは実例からの観察であり、統計的な典拠は未確認。動詞分布（do 66%）と違い、規則ではなく傾向として扱うこと。

頻出の枠（技術トーク）

枠後ろに来るもの
What this does is …この物の機能
What happens is …起きる出来事
What I'm going to do is …話者の次の行動
What I'd say is …話者の意見
All you have to do is …必要な手順（1つ）
The way this works is …仕組みの説明
The thing is …逆接・注意点
The point is …要点の再提示
It turns out (that) …意外な事実（軸なし）

挿入されるフィラー

basically / essentially / actually / obviously / literally / kind of / sort of

音節数が多くても内容語ではない。 枠の中に自由に挿入され、強勢を持たない。basically は [ˈbeɪskli] 程度まで潰れる。

産出にも使う

これらの枠は話すときの型としてそのまま使える。技術面接や設計説明で、要点を後ろに置いて強調できる。

言いたいこと枠を使った形
この関数はキャッシュするWhat this function does is it caches the result.
設定するだけでいいAll you have to do is set it up once.
でも問題があるThe thing is, it doesn't scale.
仕組みはこうですThe way this works is …
